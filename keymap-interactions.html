review.nvim Key Interaction Map

Current default key flows from the review layout. User Vim selections stay available: b and V are not review.nvim defaults.

Rail: Files / Threads Diff: unified / split Modal: popups / lists

Files Rail

Pick files, scope rows, folders, and filters.

  • <CR> opens selected file or row
  • s returns to last diff/split cursor
  • t focuses Threads

Diff / Split

Review code, add notes, navigate hunks, and toggle split.

  • s toggles unified/split here only
  • f focuses Files
  • t focuses Threads

Threads Rail

Browse local notes, remote threads, timeline, and stale items.

  • <Tab> cycles actionable rows
  • s returns to last diff/split cursor
  • <CR> opens thread or note

Notes / Blame / History

Floating workflows for queue management and context collection.

  • N opens notes list
  • B opens blame from diff
  • L opens file history
